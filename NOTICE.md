## Changes and Enhancements

This fork (Prudencia Security File Sharing / Sharecare) includes the following major changes:

- Multi-user authentication with roles (Super Admin, Admin, Regular Users, Download Accounts)
- Per-user storage quotas and management
- Enhanced download tracking with detailed user account creation
- Configurable branding system (custom logo, colors, company name)
- User and Admin dashboards with detailed analytics
- Two-mode link sharing (authenticated downloads vs. direct links)
- Enhanced user management for multi-admin scenarios
- Tailored for surveillance system customers and enterprise deployments
